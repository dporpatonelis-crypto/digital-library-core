Προσθήκη Νέου Βιβλίου (η καθημερινή ροή σου)
Κάθε φορά που θέλεις να προσθέσεις βιβλίο κάνεις μόνο 2 πράγματα:

Ανεβάζεις το νέο CSV στον φάκελο books/
Προσθέτεις μια νέα εγγραφή στο catalog.json

Η Unreal και το HTML ενημερώνονται αυτόματα χωρίς άλλες αλλαγές.


repository/
│
├── index.html              ← Κεντρικός κατάλογος βιβλίων
├── catalog.json            ← Μητρώο βιβλίων (το "μυαλό" της βιβλιοθήκης)
│
├── books/
│   ├── book_001.csv        ← Πρώτο βιβλίο (το υπάρχον σου)
│   ├── book_002.csv        ← Δεύτερο βιβλίο
│   └── ...
│
└── README.md               ← Περιγραφή repo
